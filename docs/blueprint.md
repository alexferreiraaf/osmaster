# **App Name**: OS Manager Pro

## Core Features:

- Dashboard Overview: Provides a visual summary of key metrics like total orders, pending tasks, and completed services.
- Service Order List: Displays all service orders in a sortable, searchable table.
- Create New Service Order: Form to create new service orders with fields for client info, configurations, and technical details.
- Service Order Details: Detailed view of a single service order, including client info, configurations, technical details, and service specifics.
- Order Status Update: Functionality to update the status of a service order (e.g., Pending, In Progress, Completed).
- Intelligent Technician Assignment: Use AI to suggest the best technician based on the service and location.

## Style Guidelines:

- Primary color: #4A90E2 (a moderate, bright blue), conveying trust and reliability.
- Background color: #F0F8FF (a very light, desaturated blue) provides a clean, professional backdrop.
- Accent color: #50E3C2 (a vivid turquoise-green) for highlighting interactive elements and status indicators.
- Body and headline font: 'Inter', a sans-serif font, will be used for both headlines and body text due to its modern and readable design.
- Lucide icons are used, providing a consistent and modern look throughout the interface.
- A clear, card-based layout is employed to organize information, with a sidebar for navigation and a main content area for displaying data and forms.
- Subtle transition animations on status updates and form submissions to provide feedback.
